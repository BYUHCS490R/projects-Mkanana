Box Model

Block tags are boxes that contain content. You can change the size of them. They help space and bring things together.

Inline tags are like img tags, span tags, and a tags. They are used for links etc.

Margin is the space outside the border of an element.

Border surrounds the padding and content.

Padding is the space between the content and the border inside the element, creating space within the element itself.

Different measurments that we use for ex. is percentage or %. Another on is width and height.

figure tag is a block tag that can have a picture inside. It turns the img into a black tag and not a inline tag.
